# Cognee Dashboard Specification

## Overview

A modern dashboard for Cognee - the AI Memory Engine. The dashboard enables users to upload data, build knowledge graphs, and query with AI through an intuitive interface.

---

## Design System

### Framework
- **UI Library**: shadcn/ui (React + Tailwind CSS)
- **Key Blocks**: `dashboard-01`, `sidebar-07`
- **Color Scheme**: Dark mode primary with light mode support
- **Typography**: Inter font family

## Technical Architecture

### State Management
- React Context for global state (user, cluster, theme)
- TanStack Query for server state (API calls)
- Zustand for complex UI state (debugger, logs)

### Real-time Updates
- WebSocket connection for live activity logs
- Server-Sent Events for processing status
